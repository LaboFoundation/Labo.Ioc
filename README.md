Labo.Ioc v0.1
========

A lightweight and fast inversion of control framework

Highlights
========

1. Configuration through code.
2. Constructor injection.
3. Singleton and transient lifetimes.
4. Delegate factory registration.
5. Named services registration.
6. Small in size.
7. Fast.
8. Easy to use.
